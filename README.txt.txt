Para executar o programa:
O programa recebe os seguintes parâmetros por linha de comando:
	<nomeMetodo> (crc ou checksum) <tamanhoPacote> <numeroPacotes> <seed> <probabilidade> <polinomio (hexadecimal)>
Se for usar no eclipse, siga as instruções para incluir parametros de linha de comando
	http://www.cs.colostate.edu/helpdocs/eclipseCommLineArgs.html

Ambos os métodos de detecção de erros foram construídos baseados nas implementações aprendidas em aula
e foram feitos testes para conferir o funcionamento de tais métodos

Os valores exemplo utilizados no CRC foram obtidos no site: http://www.sunshine2k.de/coding/javascript/crc/crc_js.html

Enquanto os valores utilizados para teste do Checksum foram baseados em cálculos manuais
